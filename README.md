# Peminjaman E205

## Requirements
* Composer
* NPM
* Database MySQL
## Installation
1. Run `composer install`
2. Run `npm install`
3. Run `npm run dev`
## How To Use
1. Create your own env file
2. Run `php artisan serve`
3. Access [localhost:8000/inventory](localhost:8000/inventory) or [http://localhost:8000/peminjaman](localhost:8000/peminjaman)
